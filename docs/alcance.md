# Alcance MVP

## Alcance del MVP

- Registro y visualizacion de gastos planificados.
- Registro y visualizacion de gastos reales.
- Vista base de desviacion de gastos.
- Analisis de gastos a nivel basico.
- Dashboard inicial con indicadores resumidos.

## Fuera de alcance del MVP

- Login.
- Auditoria.
- Roles y permisos.
- Flujos de aprobacion.
- Adjuntos.
- Importacion y exportacion avanzada de Excel.
- CRUD completo en esta etapa de fundacion tecnica.

## Pantallas MVP

- Dashboard basico.
- Gastos planificados.
- Gastos reales.
- Analisis y desviacion de gastos.
